# Frontend

The frontend is a single-page application built with Vite, React and TypeScript.  
It uses the Supabase API to fetch and filter card data.

## Tech stack

- Vite
- React + TypeScript
- TanStack Router
- TanStack Query
- TanStack Table
- TanStack Form
- Zod
- Shadcn UI
- Tailwind CSS
- Supabase
